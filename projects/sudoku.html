Sudoku Solver

This sudoku solver was created using Python. It uses a backtracking algorithm that tries a solution and modifies it if not valid. Seems like a long and complicated way to solve it, but it's the easiest way. Once the first few numbers are found, the solution is easily found.
Find the source code here: https://github.com/antonpp11/Sudoku

"Solve your easy sudoku in 83 milliseconds. The difficult one could take up to 0.6 seconds!"
