004 : Scroll changer

With the lockdown and everything that happened recently, I started working from home. When working from home, I would either use the laptop on my lap, or close the lid, plug it in to a monitor and use an external mouse and keyboard. The problem with this workflow is the following: when scrolling with the trackpad on my Macbook, I like to use natural scrolling, which mimics scrolling on a touch screen. But when I use my mouse, I want to use what they call inverted scrolling: Scrolling down will bring the screen down. I found the process of changing the crolling direction very complicated. I needed to navigate to System preferences, then go to the Mouse section, and then change the scrolling diretion. Here is why I created this little tool. My scripts stay in the menu bar, so in two little clicks I can change my scrolling direction. For me, this was a life saver. For installation instructions, check out the README.md in the source code.
Find the source code here: https://github.com/antonpp11/ScrollChangerMac
